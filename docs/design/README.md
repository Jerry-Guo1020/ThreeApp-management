# ThreeApp 管理平台 Tailwind UI 审核稿

这份设计稿先用于确认方向，不进入代码实现。

- 设计文件：`admin-platform-ui-review.svg`
- 画布尺寸：4200 x 4200
- 可导入 Figma：在 Figma 中使用 Import 选择该 SVG 文件即可作为设计稿查看和编辑
- 当前覆盖模块：登录、Tailwind 后台壳、工作台、酒水商品、特产商品、商品更新、评论管理、评论详情、内容配置、素材管理
- 当前包含状态：新增酒水商品抽屉、新增特产商品抽屉、新增更新弹窗、评论回复弹窗、删除评论确认、内容配置弹窗、素材上传弹窗
- 商品管理调整：酒水和特产分开管理，匹配后续分表设计
- 商品详情调整：详情介绍统一使用 AI 生成详情长图/详情图，后台只管理图片上传、排序、上下架和小程序预览
- 评论管理调整：评论默认前台展示，后台按商品进入评论详情，支持查看全部评论、二级评论、回复、置顶、删除
- 内容配置展开：首页 Banner、特产模块、酒水模块、冷库项目内容均有独立配置 UI

确认方向后，再把布局拆成 Vue 页面和组件。
